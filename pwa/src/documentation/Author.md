# Author Resource Documentation

## Overview
The Author resource allows you to manage author information in your system.

## Fields

- **Name**: The author's full name
- **Bio**: A brief biography of the author
- **Birth Date**: The author's date of birth
- **Death Date**: The author's date of death (if applicable)
- **Is Alive**: Boolean indicating if the author is currently alive
- **Next Meeting**: Scheduled meeting date with the author

## Custom Features

This resource uses a custom list component that displays authors in a specialized format.

## Tips

- Use the date pickers to easily select birth and death dates
- The "Is Alive" field is automatically calculated based on the death date
- You can filter authors by date ranges using the filter panel
