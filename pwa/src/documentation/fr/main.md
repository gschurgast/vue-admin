# Bienvenue sur WebfactoAdmin

Cette application vous permet de gérer vos ressources efficacement.
Utilisez le menu de navigation à gauche pour basculer entre les différentes ressources.

## Navigation

Utilisez la barre latérale pour naviguer entre les ressources. Vous pouvez réduire la barre latérale pour plus d'espace.

## Recherche

Appuyez sur **⌘K** (Mac) ou **Ctrl+K** (Windows/Linux) ou cliquez sur l'icône de recherche pour ouvrir la palette de commandes.
Vous pouvez rechercher des ressources et naviguer rapidement.

## Gestion des éléments

- **Créer:** Cliquez sur le bouton + dans la barre supérieure.
- **Modifier:** Cliquez sur l'icône crayon sur un élément.
- **Supprimer:** Cliquez sur l'icône corbeille sur un élément.
