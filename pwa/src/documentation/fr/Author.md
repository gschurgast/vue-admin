# Documentation de la ressource Auteur

## Aperçu
La ressource Auteur vous permet de gérer les informations sur les auteurs dans votre système.

## Champs

- **Nom**: Le nom complet de l'auteur
- **Biographie**: Une brève biographie de l'auteur
- **Date de naissance**: La date de naissance de l'auteur
- **Date de décès**: La date de décès de l'auteur (le cas échéant)
- **Est vivant**: Booléen indiquant si l'auteur est actuellement vivant
- **Prochaine réunion**: Date de réunion prévue avec l'auteur

## Fonctionnalités personnalisées

Cette ressource utilise un composant de liste personnalisé qui affiche les auteurs dans un format spécialisé.

## Conseils

- Utilisez les sélecteurs de date pour sélectionner facilement les dates de naissance et de décès
- Le champ "Est vivant" est automatiquement calculé en fonction de la date de décès
- Vous pouvez filtrer les auteurs par plages de dates à l'aide du panneau de filtres
